# DMS Relay Test

A server that receives transactions and then stores it in a blockchain

## Install NodeJS
https://nodejs.org/en/download

## Install yarn

```shell
npm install -g yarn
```

## Install Program

```shell
git clone https://github.com/MichaelKim20/dms-relay-test.git
cd dms-relay-test
yarn install
cp env/.env.sample env/.env
yarn build
```

## Add Shop

```shell
yarn run add_shop
```
